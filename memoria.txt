MEMORIAS NÃO VOLÁTEIS X VOLÁTEIS

Memorias não voláteis - Armazena mesmo desligado.

Tipos:
    HD = Hard Disk - Disco rigído

        - Discos magnéticos com uma agulha. Os discos rodam em alta velocidade e a agulha serve para salvar e buscar os dados salvos no disco.

        * Pros - Grande capacidade e barato;
        * Cons - Lento e frágil.

    SSD = Solid State Drive - Unidade de Estado Sólido

        - Uma junção de chips eletrônicos (Mesma tecnologia dos pendrives e cartões SD);
        - Geralmente usado em notebook e celular.

        * Pros - Menor, Rápido, Resistente;
        * Cons - Um pouco caro e limitações de escrita (Se muito usado e alterado tem sua vida útil reduzida).

Porém elas são memórias secúndarias. Pois o computador não trabalha diretamnte com elas, ele usa a memória de trabalho (volátel) que leva o nome de memória RAM

Memorias Voláteis - Não é armazenada ao desligar.

tipos:
    RAM = Random Acess Memory - Memoria de acesso aleatorio

        - Leva esse nome pois o tempo que leva para ir do indixe 0 ao 1 é o mesmo para ir do 0 ao 10.000.
        - Essa memória é igual a uma bancada de cozinh, quanto maior mais ingredientes cabem nela sem amontoar ou precisar guardar(deletar). Quanto mais RAM mais dados o computador consegue trabalhar sme travar ou precisar que fechemos algo.
        - Ela guarda os dados que estamos trabalhando.
        - Quando desligamos o pc os dados são apagados.
        - A memoria RAM pode ser interpretada com a seguinte tabela com indixes:
            RAM
            ----------
          1 |        |
          2 |        |
          3 |        | Cada indixe guarda 1 Byte
          4 |        |
          5 |        |
            ----------
        - Se quisermos ler/escrever algo nela usamos os indixes como endereços de onde cada byte deve ir.

        SSD / HD -----> RAM -----> CPU
        Dispensa      Bancada      Chef
    
    ROM = Read-only Memory - memoria de apenas leitura

        - É um tipo de memória com baixo armazenamento.
        - Possui varios modelos, mas todos são projetados para serem apenas lidos, sem iteração com o usuário.
        
        * Quando o computador é inicializado ele busca primeiro as informações que estão na ROM como:
            - BIOS(Basic Input/Output system), elas vão ajuda-lo a identificar os dispositivos conectados como o Hd e carregar o Sistema operacional.
